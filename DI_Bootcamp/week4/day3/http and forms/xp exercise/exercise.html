JavaScript and HTML Form Exercises

Exercise 1: HTML Form (GET)

Where will the data appear? In the URL bar as query parameters (e.g., ?username=name&message=text).

Exercise 2: HTML Form #2 (POST)

Where will the data appear? In the HTTP Request Body. To see it, open the browser's Network Tab, click the request name, and look at the "Payload" or "Form Data" section.
